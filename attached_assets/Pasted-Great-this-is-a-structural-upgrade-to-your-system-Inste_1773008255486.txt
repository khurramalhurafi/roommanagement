# 🏗 Porta Cabin Accommodation Management (System Upgrade)

## New Structure

Old System:

```
Rooms → Employees
```

New System:

```
Porta Cabin → Rooms → Employees
```

Example:

```
Porta Cabin A
   ├ Room 1
   ├ Room 2
   ├ Room 3

Porta Cabin B
   ├ Room 1
   ├ Room 2
   ├ Room 3
   ├ Room 4
```

Each **room will have its own QR code**.

| Capacity         |             |

| Status           |             |

Each **Room** will still have a QR code.

• Profile Picture
