# 3️⃣ Export QR Codes with Room Numbers

This is very important for printing & labeling.

---

## Option A: Excel Export with QR Image

Excel Sheet Columns:

| Room No | Building | Floor | QR Code Image |

QR code should appear as image inside Excel cell.

Recommended package in Laravel:

* Maatwebsite Laravel Excel package

---

## Option B: PDF QR Sheet (Recommended for Printing)

Professional format:

Page layout:

ROOM NO: 101
[ Large QR Code Image ]

ROOM NO: 102
[ Large QR Code Image ]

Each page can contain:

* 1 large QR
  OR
* 4 QR codes per page (grid layout)

This is best for printing and sticking outside rooms.

# 4️⃣ Technical Implementation (Laravel)

## Required Packages

### For Excel Export:

```bash
composer require maatwebsite/excel
```

### For PDF Export:

```bash
composer require barryvdh/laravel-dompdf
```

### For QR Code Generation:

```bash
composer require simplesoftwareio/simple-qrcode
```
